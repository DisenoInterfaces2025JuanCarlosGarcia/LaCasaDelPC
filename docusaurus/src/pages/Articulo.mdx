# Componente `<Articulo />`

El componente `<Articulo />` es un **componente presentacional y accesible** que renderiza una **tarjeta de producto** dentro del catálogo.  
Muestra de manera clara y visual: **imagen, nombre, descripción y precio** del artículo.

## Características de accesibilidad (A11y)

- La tarjeta se puede enfocar con teclado (`tabIndex="0"`).  
- Se define explícitamente el rol `article`.  
- La imagen incluye un texto alternativo (`alt`) con el nombre del artículo.  
- Se usa `loading="lazy"` para mejorar el rendimiento.  
- La estructura semántica separa claramente imagen, encabezado y contenido.

---

## Props

| Prop | Tipo | Descripción |
|------|------|-------------|
| `nombre` | `string` | Nombre del artículo; se muestra como título y se usa como texto alternativo de la imagen. |
| `precio` | `number \| string` | Precio del artículo, mostrado de forma destacada. |
| `descripcion` | `string` | Descripción breve del artículo. |
| `imagen` | `string` | URL de la imagen del artículo. |

---

## Recomendaciones de uso

- Ideal para catálogos de productos o listados visuales.  
- Se puede combinar con filtros o grids para mejorar la experiencia de usuario.  
- Compatible con temas claros y oscuros de Docusaurus.
